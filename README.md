## Simple Calculator in Python

## Description
This is a simple Python calculator that performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

The program takes two numbers as input from the user and performs the selected operation.

## Features
- User-friendly menu
- Supports decimal numbers
- Prevents division by zero
- Displays appropriate error messages for invalid choices

## Requirements
- Python 3.x

## How to Run
1. Save the program as `calculator.py`.
2. Open a terminal or command prompt.
3. Navigate to the folder containing the file.
4. Run the command:
   ```bash
   python calculator.py
   ```
5. Enter two numbers.
6. Choose an operation (1–4).
7. View the result.

## Example

```
Enter first number: 10
Enter second number: 5

Choose operation:
1. Addition
2. Subtraction
3. Multiplication
4. Division

Enter choice (1/2/3/4): 3
Result = 50.0
```

## Student Grade Calculator

## Description
This Python program calculates the total marks, average marks, and grade of a student based on marks entered for three subjects.

## Features
- Accepts the student's name.
- Accepts marks for three subjects.
- Calculates total and average marks.
- Assigns a grade based on the average.
- Displays the student's details and grade.

## Grade Criteria
- **A** : Average ≥ 90
- **B** : Average ≥ 75 and < 90
- **C** : Average ≥ 50 and < 75
- **Fail** : Average < 50

## Requirements
- Python 3.x

## How to Run
1. Save the program as `student_grade_calculator.py`.
2. Open a terminal or command prompt.
3. Navigate to the folder containing the file.
4. Run the program:
   ```bash
   python student_grade_calculator.py
   ```
5. Enter the student's name and marks for the three subjects.
6. View the total, average, and grade.

## Example

```
Enter student name: John
Enter marks for Subject 1: 85
Enter marks for Subject 2: 90
Enter marks for Subject 3: 80

Student Name: John
Total Marks: 255.0
Average Marks: 85.0
Grade: B
```

# Odd or Even Checker

## Description
This Python program checks whether a number is odd or even.

## Language
Python

## How to Run
1. Open the file in Python IDLE or VS Code.
2. Run the program.
3. Enter a number.

## Output
If the number is divisible by 2, it prints "Even".
Otherwise, it prints "Odd".
