1)# Simple Calculator in Python

## Description
This is a simple Python calculator that performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

The program takes two numbers as input from the user and performs the selected operation.

## Features
- User-friendly menu
- Supports decimal numbers
- Prevents division by zero
- Displays appropriate error messages for invalid choices

## Requirements
- Python 3.x

## How to Run
1. Save the program as `calculator.py`.
2. Open a terminal or command prompt.
3. Navigate to the folder containing the file.
4. Run the command:
   ```bash
   python calculator.py
   ```
5. Enter two numbers.
6. Choose an operation (1–4).
7. View the result.

## Example

```
Enter first number: 10
Enter second number: 5

Choose operation:
1. Addition
2. Subtraction
3. Multiplication
4. Division

Enter choice (1/2/3/4): 3
Result = 50.0
```
